ПРАКТИКА СТВОРЕННЯ АДАПТИВНОГО САЙТА

Крок 1.

Створюємо html файл, додамо до розділу < head >, тег < meta > для всіх ваших веб-сторінок. Це дозволить встановити вікно перегляду вашої сторінки, яке дасть браузеру інструкції про те, як керувати розмірами та масштабуванням сторінки.

Крок 2.

Додаємо тег <style> для визначення стилів компонентів сторінки. Робимо відступ 0 і додаємо гнучкий контейнер (display: flex та flex-wrap: wrap). Потім вказуємо ширину та мінімальну висоту (у пікселях) для елементів aside та main:
